North 将所有用户划分为三类：`普通用户`、`管理员` 和 `系统`，其权限对比如下

### 2.1.1 网页端

|          | 图片上传 | 仪表盘 | 图片管理 | 成员管理 | 插件管理 |

|  管理员  |    ✅     |   ✅    |    ✅     |    ✅     |    ❌     |
|  系 统   |    ✅     |   ✅    |    ✅     |    ✅     |    ✅     |
